GitHub Pages do Zero

Como criar sua página pessoal no GitHub, editar arquivos direto pelo navegador e publicar conteúdo — sem instalar nada no computador.

1

Criar conta no GitHub

Acesse github.com e clique em Sign up. Preencha e-mail, senha e escolha um nome de usuário.

  • O nome de usuário vira parte do endereço do seu site.
  • Exemplo: usuário luizftoledo → site em luizftoledo.github.io
  • Confirme o e-mail quando o GitHub mandar a verificação.

2

Criar o repositório do site

No GitHub, clique no + no canto superior direito → New repository.

  • No campo Repository name, digite exatamente: seunome.github.io
  • O padrão é sempre o seu nome de usuário + .github.io
  • Deixe Public marcado.
  • Ative a opção Add a README file.
  • Clique em Create repository.

4

Gerar o HTML com o Claude

Acesse claude.ai e peça para criar a página que você quer. Seja específico: nome, profissão, cores, seções.

✦ Claude · claude.ai
Crie uma página HTML de "Sobre mim" para meu site pessoal. Me chamo Luiz Toledo, sou jornalista investigativo e roteirista. Quero fundo branco, meu nome em destaque, uma bio curta e links para redes sociais.
Aqui está sua página:
<!DOCTYPE html> <html lang="pt-BR"> <head> <title>Luiz Toledo</title> ... </head> <body> <h1>Luiz Toledo</h1> ... </body> </html>
✅ Selecione todo o código gerado (Ctrl+A dentro do bloco de código) e copie com Ctrl+C.
5

Volte para o seu repositório no GitHub. Clique no botão Add file → Create new file.

6

Salvar e publicar

Clique no botão Commit changes… no canto superior direito do editor. Uma janela vai abrir.

Commit changes

Commit message
Deixe a mensagem padrão ou escreva o que preferir. Ela é só um registro do que você alterou.
Cancel Commit changes
🚀 Após clicar em Commit changes, o GitHub publica a alteração em até 2 minutos. Acesse seu site para ver o resultado.
7

Adicionar novas páginas

Repita os passos 4 e 5 para cada nova página. Peça ao Claude o HTML de uma nova seção e crie um arquivo com nome diferente.

Arquivo
index.html

luizftoledo.github.io/sobre.html

luizftoledo.github.io/contato.html
⚠️ O arquivo principal do site deve sempre se chamar index.html. É ele que o GitHub carrega quando alguém acessa o endereço raiz.
8

Editar uma página existente

No repositório, clique no arquivo que quer editar. No topo do arquivo, clique no ícone de lápis ✏️ para abrir o editor.

github.com/luizftoledo/luizftoledo.github.io/blob/main/index.html

Após editar, clique em Commit changes… e confirme. A atualização vai ao ar em instantes.
