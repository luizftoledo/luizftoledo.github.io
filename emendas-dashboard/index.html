Metodologia

Fonte primária para a série diária: CSV 2026_EmendasParlamentaresPorDocumento.zip (CGU/Portal da Transparência), agregado por data do documento de empenho. Esta data é a do registro fiscal — o que dá granularidade real dia a dia.

Fonte para o teto autorizado e cross-check: SIOP (Sistema de Planejamento, Orçamento e Finanças do Governo Federal), painel "Emendas Parlamentares — Resultado Primário 6/7/8/9" (Qlik/SigaBrasil). Capturado via Selenium — base SIAFI atualizada com defasagem de até 2 dias úteis.

Definição: "empenhado" é o valor já reservado para pagamento (compromisso firmado pela União), não o que foi efetivamente desembolsado ("pago").

Atualização: rotina diária às 13:10 UTC (10:10 BRT) via GitHub Actions. Quando a fonte original não publica novo arquivo no dia, o pipeline registra source_changed: false e mantém o snapshot anterior.

Fontes oficiais

  • Portal da Transparência — base UNICO de emendas (CGU)
  • Portal da Transparência — documentos por emenda (CSV anual)
  • SIOP — Painel do Orçamento Federal (Planejamento)
  • Siga Brasil — Senado (referência)
